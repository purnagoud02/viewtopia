# Folder Structure

- client/: React frontend application
  - src/components/: reusable UI components
  - src/pages/: route-level screens
  - src/context/: state/context providers
  - src/services/: API wrappers and helpers
- server/: backend application
  - config/: database and upload configuration
  - controllers/: route handlers
  - middleware/: auth, error, and request protection
  - models/: Mongoose schemas
  - routes/: API endpoints
  - uploads/: generated media files
- docs/: documentation for the project
- database/: seed and sample data
